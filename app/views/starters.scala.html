Chat

First, be sure to join the #vindinium IRC chan on Freenode if you have any questions or if you want to geek around a little bit. Here is a direct link for people who don't have an IRC client.

Registration

First, you will need to register your AI (Artificial Intelligence) on the server using the register page. You just have to enter a name and we will give you a secret key. Be sure to write it down as there is no way to recover it later. This key will be used to uniquely identify your bot with the name you choosed.

Once you have your key, you can already try one of the starter packs in training mode if you want. More on modes in the next section.

API

Game modes

The game server supports two modes: training and arena. Both have different urls.

Training

In training mode, you can play against some dummy bots, you can specify the number of turns you want to play and, if you want, you can provide a specific map to test your algorithms. The games played in Training mode are not scored, so feel free to use it as much as you want.

In training mode, the url you will need to use is: 

http://vindinium.jousse.org/api/training

Arena

In arena mode, the url you will need to use is: 

http://vindinium.jousse.org/api/training
